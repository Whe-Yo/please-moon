# 3-way 비교 — SpaceX 조사 (260622)

**질문**: "주식 분석은 Antigravity(Gemini) 단독으로 충분한가?" → Claude단독 vs Gemini단독 vs clemini 실측.
**대상**: SpaceX (시점에 민감한 사실이 결정적인 케이스).

## 결과

### ⓐ Gemini 단독 (agy --print, 1콜)
- 산출: 구조적·풍부함 — 기업가치, 노출 경로(세컨더리/DXYZ/우회), 촉매·리스크, 신호. 형식 훌륭.
- **치명적 오류**: SpaceX를 **"비상장, ~$210–350B, Starlink 분할 IPO 불확실"**로 단정. → **2026 IPO를 통째로 놓침.** 훈련 지식 기반, 라이브 검증 없음. 그럴듯하게 **자신 있게 틀림**.
- 비용: ~25k 토큰(Ultra 쿼터), 빠름. 단 시점민감 사실에 위험.

### ⓑ Claude 단독 (WebSearch + 종합)
- 산출: 라이브 검색으로 **현재 사실 포착** — 2025.12 ~$800B 텐더, 2026 IPO(집계매체: 06-12 상장 SPCX ~$1.75T), DXYZ는 NAV 대비 큰 프리미엄, 리테일 배정 확대. 결정적 사실 정확.
- 한계: 구조적 랜드스케이프(노출경로·촉매 분류)는 내가 처음부터 쌓아야 함(시간·토큰). 출처 품질도 직접 가려야 함(일부는 CFD·AI블로그).
- 비용: Claude 검색+종합(Claude 예산 소모).

### ⓒ clemini (Gemini 조사 + Claude 검증·종합)
- Gemini의 **구조적 breadth**(노출 경로·촉매·리스크 — DXYZ 프리미엄·세컨더리·Starship 촉매·규제 리스크는 **여전히 유효**) + Claude의 **검증**(스테일한 중심 사실을 잡아 교정: 비상장 → 2026 IPO).
- 최선 산출: Gemini 뼈대 + Claude 사실교정 + 출처 신뢰도 hedge.
- 비용: ~25k Gemini(Ultra) + 소량 Claude 검증 → **Claude-light, 최고 ROI**.

## 판정 — "Gemini 단독으로 충분한가?"
**아니다 (주식엔 특히).** 시점민감·사실결정적 도메인에서 Gemini 단독은 **자신 있게 틀릴** 수 있다(IPO 누락이 그 증거). 
- **올바른 구조 = clemini**: Gemini가 넓은 뼈대·노출경로·시나리오를 싸게 깔고, **Claude가 시점민감 사실을 라이브 검증·교정**. (패턴 A+D 정확히 일치.)
- 단, Gemini의 **구조·분류·시나리오 가치는 실재**했다 — 버리는 게 아니라 검증 위에 얹는다.

## 에이전트 설계 함의
주식 에이전트는 **clemini 구조 필수** + 규칙: ① 시점민감 사실(가격·상장상태·실적·일정)은 **항상 Claude 라이브 검증** ② 출처 신뢰도 표기 ③ **매수/매도 권유 금지** — 정보·시나리오·리스크만, 면책 명시.
