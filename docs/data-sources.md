# 신뢰 데이터 소스 + 소스 신뢰도 등급 (KR + US)

> 방법: Gemini 벤치마크 오프로드 + Claude 검증(한국 공식 API URL 실재 확인). please-moon은 **정보형(매매 안 함)** → 시세·재무·공시 데이터가 핵심, 주문 API 불필요.

## 소스 신뢰도 등급 (fact-check 틀)
| Tier | 신뢰도 | 예시 | 에이전트 행동 |
|---|---|---|---|
| **T1 공식** | 절대 | DART·KRX·SEC EDGAR·거래소 | 수치의 **단일 팩트**. 타 소스와 다르면 T1로 덮어씀 |
| **T2 평판매체·브로커API** | 높음 | Reuters·Bloomberg·WSJ·연합인포맥스, KIS·토스 API | 속보·시장팩트 채택. 단 수치는 T1 공시 나오면 교차·갱신 |
| **T3 표준화 데이터·리서치** | 중간 | FMP·Alpha Vantage·증권사 리포트 | 시계열·지표 수집용. **목표주가/투자의견은 '컨센서스(의견)' 변수**로만 |
| **T4 커뮤니티·AI요약·블로그** | 낮음 | Reddit·X·Stocktwits·AI 콘텐츠팜 | **사실근거 사용 불가.** 심리(sentiment) 지표로만 격리 |

## T1 — 1차/공식 (무료, 최우선)
- **한국**
  - **OPENDART** `opendart.fss.or.kr` — 사업·분기보고서, 재무제표, 주요사항(증자·합병), 지분공시. 공식 무료 API. *(검증됨)*
  - **KRX OpenAPI / KIND** `openapi.krx.co.kr`, KIND(공시·투자주의/관리종목·공매도). *(검증됨)*
- **미국**
  - **SEC EDGAR** — 10-K/10-Q/8-K, Form 3/4/5(내부자), 13F(기관 보유), S-1. 무료 API.

## T2/T3 — 프로그램용 데이터 API (please-moon 후보)
| API | 비용 | 커버리지 | 용도 |
|---|---|---|---|

| **토스 OpenAPI** | 무료(사전신청) | KR+US | 시세(우리가 쓸 카테고리). 키=사용자 발급 |
| **FMP** | 무료/유료 | US 최적·KR 기본 | SEC 10-K/Q 파싱한 표준 재무제표 |

## 뉴스·분석 (T2, 맥락용)
Reuters · Bloomberg · WSJ/FT · **연합인포맥스**(한국 표준). 대부분 유료/페이월 → 맥락 파악용, **수치는 T1로 검증**.

## 피해야 할 것 (T4)
AI/SEO 콘텐츠팜(intellectia·ts2 류), CFD·마진 브로커 '가이드'(레퍼럴 유도), 미인가 리딩방·찌라시, 수익인증 유사투자자문. → SpaceX 조사 때 실제로 섞여 나왔음.

## 교차검증 규칙 (Gemini 안티테제 반영)
1. **시차**: 속보(T2)로 잡은 사실은 'T1 공시 확인 대기'로 표기 → 공시 파싱되면 확정·보정.
2. **한국 데이터 왜곡**: 무료 글로벌 API의 한국 **수정주가·재무**는 오류 잦음 → **OPENDART/KRX와 교차**, 편차 시 alert.
3. **루머**: 미공개·M&A 소문은 수치 DB 반영 금지, sentiment로만.

## please-moon 적용
- **사실(가격·재무·공시)**: T1(OPENDART/EDGAR/KRX) + T2 브로커API(KIS/토스). 항상 **Claude 라이브 검증**(시점민감).
- **맥락**: T2 매체. **의견(목표가)**: T3 컨센서스 변수.
- **출력엔 출처 등급 표기**(SpaceX 분석에서 한 것처럼: "집계매체" vs "공식").
- 시작: 무료(FinanceDataReader/EDGAR/OPENDART)로 골격 → 토스/KIS 키 나오면 어댑터 스왑.
